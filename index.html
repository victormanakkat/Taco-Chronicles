About

The Taco Chronicles is a single player action adventure game. You are Dr. Taco, a scientist obsessed with tacos. The goal of the game is to get to Taco Bell safely, without getting shot by crazy monsters that somehow inhibit the game. To earn points, you eliminate monsters, eat tacos, and collect supplies. Tacos give you 7 points, monsters give you 100 points, and supplies give you 25 points. As your total score increases, you unlock various weapons from a 9mm pistol to a flamethrower. Currently there is only one level, but more will be developed soon. Read on to install and play your own copy of the Taco Chronicles.

Install

To play the Taco Chronicles, you will first need to download a copy onto a computer. If you have some computer experience, follow the seven steps below. If you don't, please read this INSTALL.pdf file.

  1. Download zip from GitHub
  2. Move zip file to safe location.
  3. Extract zip file.
  4. Make a shortcut of the exe file to desktop.
  5. Play the game.
